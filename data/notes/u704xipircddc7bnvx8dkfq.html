Autoformating for code

Visual studio provides a package "Black formatter" for python, which will automatically reformat when you press Ctrl. + Shift + I (for Linux) or Ctrl. + Shift + F (for Windows). If you change the file "Open Users Settings (JSON)" and you add the line "editor.formatOnSave": true into the barely brackets, it will format if you save the file.
