# [What is Feature Selection? (IBM)](https://www.ibm.com/think/topics/feature-selection)
- **Feature selection:** Process of selecting most relevant features of a dataset for building and training a model. Reducing the features improves model performance (e.g. from all 118 elements to the handful used in bio chemistry), and lowers computational demand. 
- **Feature:** refers to the individual measurable property/characteristic of a data point. The attribute of the data that describes what is being observed. 
- **[Feature engineering](https://www.ibm.com/think/topics/feature-engineering)**: the act of preparing data and curating a feature set to be used for ML.
